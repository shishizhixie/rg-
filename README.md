# 人格核心插件 `astrbot_plugin_personality_core`

让 AI 拥有情绪、个性、好感度和内心思考。融合 FavorPro 的 LLM 自评机制。

---

## 功能

| 模块 | 说明 |
|------|------|
| **20维情绪系统** | 快乐/悲伤/愤怒/恐惧/惊讶/喜爱/厌烦/期待/暧昧/嫉妒/羞耻/愧疚/轻蔑/同情/淫乱/怀疑/失望/孤独/感激/释然（0-100），指数衰减+关键词微调+LLM自评 |
| **好感度系统** | 每个用户独立的好感度(-100~100)、印象描述、关系等级，LLM 自评更新 |
| **回答规划** | LLM 先内心思考再回复，思考内容可显示/隐藏 |
| **行为映射** | 好感度和情绪影响回复风格（冷淡/热情/病娇/攻击等） |
| **会话隔离** | 可选：每个群聊/私聊的好感度独立计算 |

## 工作流程

```
用户消息
  → on_llm_request: 注入情绪 + 好感度 + 行为映射 + 思考要求
  → LLM 输出:
       【思考】内心规划
       【回复】实际回应
       [Emotion: {...}]  [Favour: x, Attitude: y, Relationship: z]
  → on_llm_response:
       ① 提取【思考】→ 日志记录 / 可显示给用户
       ② 解析 [Emotion] → 更新情绪
       ③ 解析 [Favour] → 更新好感度 → 替换为用户可见格式
       ④ 清理标签 → 只发回复到 QQ
```

情绪受三方面影响：
- **关键词匹配**（快速响应）
- **LLM 自评**（精准修正，通过回复标签）
- **指数衰减**（随时间回归默认值）

## 安装

1. 将 `astrbot_plugin_personality_core` 目录放入 `data/plugins/`
2. 在 WebUI → 插件管理 → 启用插件并配置
3. 重启 AstrBot

## 配置

在 WebUI 插件管理页可配置或使用命令：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `enabled` | bool | true | 启用/禁用插件 |
| `session_based` | bool | false | 会话独立好感度 |
| `show_thinking` | bool | false | 显示内心思考到回复中 |

## 命令

| 命令 | 说明 | 用法 |
|------|------|------|
| `人格状态` | 查看情绪 + 好感度 | - |
| `人格开关 on/off` | 启用/禁用插件 | `人格开关 off` |
| `思考开关 on/off` | 显示/隐藏内心思考 | `思考开关 on` |
| `情绪重置` | 重置情绪到默认值 | - |
| `查询好感 [user_id]` | 查看用户好感度 | `查询好感` 或 `查询好感 123456` |
| `重置好感 [user_id]` | 重置用户好感度为默认 | `重置好感 123456` |
| `好感排行 [num]` | 好感度 TOP 排行 | `好感排行 5` （默认10） |
| `负好感排行 [num]` | 负好感度排行 | `负好感排行 5` |
| `人格帮助` | 显示帮助信息 | - |

## 好感度行为映射

| 好感度 | 态度 | 风格 |
|--------|------|------|
| ≥ 90 | 病娇 | 偏执占有，控制欲极强 |
| 75~89 | 热情亲密 | 主动撒娇 |
| 40~74 | 友好积极 | 友善互动 |
| 0~39 | 中立客气 | 保持距离 |
| -1~-39 | 冷淡敷衍 | 不想理人 |
| -40~-79 | 尖锐刻薄 | 厌恶敌意 |
| ≤ -80 | 攻击模式 | 威胁恶意 |

情绪阈值（如愤怒 > 70、孤独 > 65 等）会追加到回复规则中进一步影响行为。

## 数据文件

存储在 `data/plugin_data/astrbot_plugin_personality_core/`：

| 文件 | 说明 |
|------|------|
| `emotion_state.json` | 当前情绪状态（跨重启持久化） |
| `favor_data.json` | 所有用户的好感度数据 |
| `persona.json` | 角色设定（仅内部使用，人格由 AstrBot 配置） |
| `config.json` | 插件配置 |

## 调试

查看日志中的思考过程：

```bash
docker logs astrbot | grep '💭\|📊\|🧠\|⚠️'
```

输出示例：
```
🧠 注入思考 | 情绪={"joy":50,...} | 好感=0
💭 内心思考: 她今天心情看起来不太好
📊 情绪自评: {"joy":45,"sadness":60,...}
📊 好感自评: user=2106979549 favour=-25 attitude=怪人
```

## 兼容性

- 需要 AstrBot >= 4.16
- 如果同时安装了旧版 `favorpro` 插件，请禁用以避免冲突
